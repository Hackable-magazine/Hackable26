Sources et fichiers pour le magazine Hackable #26
=================================================

Ce dépôt contient les croquis, sources et fichiers utilisés dans le 26ème numéro de [Hackable](http://www.hackable.fr/) :

* `espwifieeprom` : croquis Arduino pour ESP8266 stockant les informations en EEPROM émulée
* `espmqtt_base` : croquis du client MQTT de base (publication et abonnement)
* `espmqtt_auth` : croquis du client MQTT avec authentification par mot de passe
* `espmqtt_authTLS` : croquis du client MQTT avec chiffrement SSL/TLS et authentification par mot de passe
